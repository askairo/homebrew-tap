# homebrew-tap

English | [中文](README_ZH.md)

Homebrew tap for Askairo desktop and CLI tools.

## Install Tap

```bash
brew tap askairo/tap
```

## Install clicky

```bash
brew install clicky
```

## Upgrade

```bash
brew update
brew upgrade clicky
```

## Uninstall

```bash
brew uninstall clicky
```

## Maintainers

- Cask definitions are stored under `Casks/`.
- `Casks/clicky.rb` is updated from the clicky release pipeline.
