# homebrew-tap

[English](README.md) | 中文

Askairo 桌面工具与 CLI 工具的 Homebrew tap 仓库。

## 安装 Tap

```bash
brew tap askairo/tap
```

## 安装 clicky

```bash
brew install clicky
```

## 升级

```bash
brew update
brew upgrade clicky
```

## 卸载

```bash
brew uninstall clicky
```

## 维护说明

- Cask 定义存放在 `Casks/` 目录。
- `Casks/clicky.rb` 由 clicky 发布流水线更新。
